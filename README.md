# HyperPlannerTextExtraction
Extraction of Transcript PDF Text for HyperPlanner

Technologies used:
- AWS Textract
- AWS Simple Storage Service (S3)
- AWS Lambda
- AWS Simple Notification Service (SNS)
- AWS Simple Queue Service (SQS)
